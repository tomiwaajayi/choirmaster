# @choirmaster/cli

The `choirmaster` command. Runs plans through the [ChoirMaster](https://github.com/tomiwaajayi/choirmaster) orchestrator.

## Install

```bash
npm install -g @choirmaster/cli
```

## Usage

```bash
choirmaster --help
```

See the [main README](https://github.com/tomiwaajayi/choirmaster#readme) for full docs.
